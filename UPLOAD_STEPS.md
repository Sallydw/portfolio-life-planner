# Quick Upload Steps

Since you already have a GitHub repo connected, here are the commands:

```bash
# 1. Add all changes
git add .

# 2. Commit with a message
git commit -m "Update portfolio: Add professional README, LICENSE, and code improvements"

# 3. Push to GitHub
git push origin main
```

That's it! Your changes will be uploaded to:
https://github.com/Sallydw/portfolio-life-planner
